# react-auto-deploy-template

此项目为react多页面项目配置的模板，已经构建了基础设施。

**node版本：v10.6.0**

**npm版本：v6.8.0**

### 特点

1. 前后端开发分离，并行开发
2. webpack配置模块化并集中于webpack
3. 固定的项目目录，有利于项目的统一性
4. 项目走流水线不需要额外配置
5. 不需要担心依赖版本问题
